## What’s new and why it matters
What is FaceAuth? FaceAuth is a lightweight face authentication daemon for Linux that automatically unlocks your screen using your face. Features Automatic IR sensor detection with RGB webcam fallback Camera activates only when screen is locked Auto unlocks on face recognition Works on Fedora, Ubuntu and Arch Linux Simple one command installation How It Works FaceAuth runs a background daemon that monitors screen lock events.
