I was tired of it. So I built projectpulsewire — and it hit 23 GitHub stars in its first month. What is projectpulsewire? It's a Python CLI tool that gives you instant access to 47 premium EQ presets and 404 IRS (Impulse Response) files for EasyEffects on Linux — all installable in a single command.
