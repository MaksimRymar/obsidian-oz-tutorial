## What’s new and why it matters
I needed business phone numbers for a sales project. Google Places wanted $0.03 per lookup. No thanks. Built this over a weekend. Search "plumber in New York" → returns: Business name Phone number Address Website GPS coordinates 60+ business types supported. Every country with OpenStreetMap data. Stack: FastAPI + Render free tier + Overpass API Total cost to run: $0/month. Free tier: 3,000 requests/month. Paid plans from $29.99/month. Search "Places Search API" on RapidAPI to try it. Code on GitHub: github.com/buigialy30206-eng/google-maps-api
