## What’s new and why it matters
There is a difference between asking AI to write SQL and giving AI a controlled way to use a database. The first is an assistant. The second is infrastructure. Teams mix these up all the time. You review it, fix joins, run it, inspect the result, and decide whether it makes sense. That can save time. But the human is still the database operator.
