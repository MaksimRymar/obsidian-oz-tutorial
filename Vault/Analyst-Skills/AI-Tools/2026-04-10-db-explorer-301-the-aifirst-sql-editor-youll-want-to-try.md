## What’s new and why it matters
Developers spend a surprising amount of time writing SQL, debugging queries, and navigating schemas. DB Explorer 3.0.1 aims to change that. This release introduces a fully schema‑aware AI SQL Generator with support for OpenAI, Claude, DeepSeek, Gemini, and custom endpoints. Other highlights: Multi‑database support Execution plan viewer Health dashboard Multi‑tab SQL editor 13 themes JVM‑bundled builds If you work with databases daily, this tool is worth exploring. We’re actively collecting feedback to shape the next version. https://sourceforge.net/projects/db-explorer/
