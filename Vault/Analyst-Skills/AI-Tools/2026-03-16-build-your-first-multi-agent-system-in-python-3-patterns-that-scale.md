## What’s new and why it matters
Your single AI agent handles 5 tools. Then 10. Then 20. Somewhere around tool number 15, it starts picking the wrong one half the time. This is the single-agent ceiling. Every production team hits it. The fix is not a better prompt — it is splitting work across multiple specialized agents that coordinate. Multi-agent systems sound complex. They are not. If you have built one agent, you already know 80% of what you need. The remaining 20% is coordination — and LangGraph gives you 3 patterns to handle it.
