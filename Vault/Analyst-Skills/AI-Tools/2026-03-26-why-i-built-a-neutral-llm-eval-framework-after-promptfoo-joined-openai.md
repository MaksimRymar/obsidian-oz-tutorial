## What’s new and why it matters
A few weeks ago, Promptfoo — one of the most popular open-source LLM evaluation frameworks — joined OpenAI. I don't think that's inherently bad. That's a conflict of interest that matters. So I built Rubric — an independent, MIT-licensed LLM and AI agent evaluation framework. No corporate parent. Open source forever.
