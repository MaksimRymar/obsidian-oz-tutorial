## What’s new and why it matters
What are DDL and DML? DDL (Data Definition Language) DDL refers to SQL commands used to define and manage the structure of a database. These commands control how data is stored. Examples: 1.CREATE – used to create tables or databases 2.ALTER – used to modify existing structures 3.DROP – used to delete tables or databases Table structure after executing CREATE statement.
