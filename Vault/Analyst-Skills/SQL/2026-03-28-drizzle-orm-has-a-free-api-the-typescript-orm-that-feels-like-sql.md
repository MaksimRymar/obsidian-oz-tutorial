## What’s new and why it matters
What is Drizzle ORM? Drizzle ORM is a TypeScript ORM that embraces SQL instead of hiding it. Unlike Prisma that has its own query language, Drizzle's query builder maps 1:1 to SQL — if you know SQL, you already know Drizzle. Why Drizzle Over Prisma?
