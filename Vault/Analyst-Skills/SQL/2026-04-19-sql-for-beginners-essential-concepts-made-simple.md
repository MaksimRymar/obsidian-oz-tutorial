## What’s new and why it matters
SQL stands for Structured Query Language. It is the standard language used to communicate with databases. Think of SQL as the "English" you speak to a database when you want to get information, add new data, update records, or delete something. SQL allows you to do 4 main things (often remembered as CRUD) where C represents Create, R represents READ while U represents Update and D represents Delete.
