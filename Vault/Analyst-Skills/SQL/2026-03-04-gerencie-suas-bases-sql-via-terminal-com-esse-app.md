## What’s new and why it matters
Para desenvolvedores, DBAs e analistas que precisam de uma ferramenta leve, rápida e poderosa para diferentes bancos de dados no terminal. Pam é uma ferramenta de linha de comando (CLI) minimalista, escrita em Go , projetada para facilitar o gerenciamento e a execução de consultas SQL em múltiplos bancos de dados. Inspirada no universo de “The Office”, Pam homenageia a personagem que organiza tudo com eficiência. Principais Destaques Biblioteca de Queries: Salve e organize suas consultas SQL mais usadas, facilitando o acesso e reuso. Multi-banco de dados: Compatível com PostgreSQL, MySQL, SQLi…
