## What’s new and why it matters
What is a database. A database is an organized collection of data stored electronically. What is SQL SQL stands for structured query language. It is used for storing data, retrieving data, updating data and deleting data. Schema in SQL A schema is a logical container or namespace inside a database that groups related objects like tables, views, indexes, stored procedures, functions and sequences. You can think of a schema as a folder inside a database helping you organize and manage objects. Why use schemas. Schemas help organize large datasets into sections, control access to different parts…
