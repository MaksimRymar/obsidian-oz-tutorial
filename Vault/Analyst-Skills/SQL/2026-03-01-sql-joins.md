## What’s new and why it matters
Joins are used to combine rows from two or more tables based on a related column between them. This allows you to retrieve connected data stored across multiple tables in a single result. Types of joins. 1: INNER JOIN INNER JOIN returns only matching rows from both tables. If there is no match between the tables the row is excluded from the result. It is used when you only want records that exist in both tables or want to combine related data. 2: LEFT JOIN LEFT JOIN returns all rows from the left table and matching rows from the right table.
