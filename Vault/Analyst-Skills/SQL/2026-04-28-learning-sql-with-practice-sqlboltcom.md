## What’s new and why it matters
Learning SQL (Day 1). Keeping it practical. 3 core things I learned today: Filtering (WHERE) SELECT city FROM north_american_cities WHERE country = 'Canada'; Sorting (ORDER BY) SELECT city FROM north_american_cities ORDER BY population DESC; Limiting results (LIMIT + OFFSET) SELECT city FROM north_american_cities ORDER BY population DESC LIMIT 2 OFFSET 2; Key insight: SQL is not hard — translating the problem correctly is the real skill.
