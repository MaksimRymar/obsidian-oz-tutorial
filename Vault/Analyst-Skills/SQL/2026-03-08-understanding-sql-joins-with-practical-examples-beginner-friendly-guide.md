## What’s new and why it matters
In real world databases, data is rarely stored in a single spreadsheet. Instead it is organized across multiple related tables. SQL provides powerful tools such as joins to combine and analyze this data efficiently. What is SQL Joins. SQL joins are used to combine data from two or more tables based on a related column between them. In most relational databases, information is stored across multiple tables and joins allow us to bring this data together for analysis.
