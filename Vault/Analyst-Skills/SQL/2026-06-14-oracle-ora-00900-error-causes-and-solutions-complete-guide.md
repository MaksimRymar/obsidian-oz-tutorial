ORA-00900: Invalid SQL Statement — Causes, Fixes & Prevention ORA-00900 is thrown by Oracle when the database engine cannot recognize the submitted text as a valid SQL statement. This typically happens due to typos in SQL keywords, unsupported syntax, or malformed dynamic SQL strings.
