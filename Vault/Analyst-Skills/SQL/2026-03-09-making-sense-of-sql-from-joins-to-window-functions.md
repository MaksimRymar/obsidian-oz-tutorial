This is where JOINs and Window Functions shine. Window function A window function performs a calculation across a set of table rows that are related to the current row. Window functions can be compared to aggregate functions but unlike aggregate functions window functions does not cause rows to be grouped into a single output row that is the rows maintain their original identities.
