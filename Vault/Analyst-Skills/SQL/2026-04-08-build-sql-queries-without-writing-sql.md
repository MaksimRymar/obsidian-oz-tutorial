## What’s new and why it matters
Not everyone wants to write SQL by hand. Sometimes you just want to grab some data from a couple of tables without remembering JOIN syntax. DBCode has a visual query builder that lets you do exactly that. How It Works Open the query builder, pick your tables. DBCode shows you the columns. Click the ones you want. Add a join by dragging between tables. Set filters with dropdowns. Pick sort order. DBCode writes the SQL. You see it update in real time as you click.
