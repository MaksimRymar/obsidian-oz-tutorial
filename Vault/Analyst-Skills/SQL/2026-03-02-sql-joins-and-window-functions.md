## What’s new and why it matters
SQL Joins and Window Functions SQL joins allows you to collate/add two or more tables together using a common identifier e.g. a foreign key. Due to normalization, one may not get all the required columns in a single table in order to reduce data redundancies. The common identifier acts as a transportation between the two tables allowing columns from both tables be included in the query result set.
