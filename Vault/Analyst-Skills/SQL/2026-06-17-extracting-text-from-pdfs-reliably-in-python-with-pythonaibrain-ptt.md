## What’s new and why it matters
PDFs are everywhere. Research papers. Reports. Invoices. Documentation. Books. Contracts. Sooner or later, almost every application needs to extract text from PDF files. The problem is that PDF files in the real world are often messy. Some are partially corrupted. Some contain problematic pages. Some fail halfway through extraction. Many extraction examples stop at: text = extract ( pdf ) but production applications need something more robust. That's why I built the PTT (PDF-To-Text) module for Pythonaibrain.
