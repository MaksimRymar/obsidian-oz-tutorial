## What’s new and why it matters
📌 DDL vs DML In SQL, commands are broadly divided into two categories: DDL (Data Definition Language): These commands define and modify the structure of database objects such as tables, schemas, and indexes. Examples include CREATE, ALTER, and DROP. DML (Data Manipulation Language): These commands deal with the actual data stored inside tables. Examples include INSERT, UPDATE, DELETE, and SELECT. The key difference is that DDL changes the structure of the database while DML changes the contents of the database.
