## What’s new and why it matters
If you are just starting out with SQL, you have probably already learned how to write a SELECT query, filter data with WHERE, and maybe even join two tables together. But what happens when you need to combine results from two completely different queries? That is exactly where SET operators come in. Think of SET operators as tools that let you stack or compare the results of two SELECT queries. Instead of joining tables side by side like a JOIN does, SET operators combine query results on top of each other, row by row.
