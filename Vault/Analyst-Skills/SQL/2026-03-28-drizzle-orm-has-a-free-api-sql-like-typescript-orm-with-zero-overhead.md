## What’s new and why it matters
What if your ORM looked like SQL — so you could think in SQL while getting full TypeScript type safety? Drizzle ORM is a TypeScript ORM that feels like writing SQL. No query builder abstraction, no magic — just typed SQL. Why Drizzle SQL-like syntax — if you know SQL, you know Drizzle Zero overhead — thin wrapper, no runtime query building Drizzle Kit — migration tool with push, pull, and studio All databases — Postgres, MySQL, SQLite, Turso, Neon, PlanetScale Edge-ready — works with HTTP-based databases (Neon, Turso) Drizzle Studio — browser-based database viewer Quick Start npm install drizz…
