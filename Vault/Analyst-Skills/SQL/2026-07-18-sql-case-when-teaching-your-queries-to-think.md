## What’s new and why it matters
Every query we've written so far has been honest about what it can do. It retrieves columns, filters rows, sorts results. It reports what's already there. Today we taught SQL to make decisions. Not retrieve a price - classify it. Not return a stock level - evaluate it. Not show an age - categorise it. CASE WHEN is the if/elif/else of SQL, and once it clicks, students stop thinking of queries as read-only reporters and start thinking of them as tools that can reason about data.
